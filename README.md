# まんぷくっ

## サービス概要
昨今大盛りグルメが流行っている印象です。某デカ盛りグルメ番組の影響でしょうか。
当サイトは大盛りグルメを取り扱う飲食店に焦点を置いた「大盛りグルメ専用SNS」です

## このテーマを選んだ理由
「大盛り　飲食店」などでWEB検索をかけると、食べログやまとめサイトが表示されます。
ですが蓋を開けてみると大盛りと関係のない飲食店が表示されたり、ファストフード店やチェーン店
が表示されることもあります。まとめサイトに至っては機能性が乏しかったりという印象です。
そして、意外にも大盛り専門で紹介するサイトがないことに気がつき、「大盛りグルメ専用SNS」
のようなものがあれば喜ばれるのではないか、という発想がこのテーマを選んだ理由です。

## ターゲットユーザー
主に学生・会社員。

## 主な機能
主な用途としてはユーザーが飲食店を紹介する投稿をしたり、
それに対していいねやコメントをしていく シンプルなSNSといったイメージです。

①お店の情報が投稿できる
* 写真が投稿できて、店構え、料理の写真が投稿できる
* お店の説明、その他感想などの説明文が投稿できる
* 場所の地図を表示する(住所だけで十分かも)

②いいね/コメント機能あり
* 閲覧した他のユーザーがいいね/コメントできる
* ブックマーク機能あり
* いいねの数で表示順を変える
* フォロー機能でお気に入りのユーザーをフォローできる

③検索機能あり
* エリアを指定/検索できる
* ジャンルが指定/検索できる
* お店の名前で検索できる
